 Student Attendance System
 Description

It is a simple Python program to manage the students attendance system by using some basic OOP concepts.

 Features
Add student
View students
Update student
Calculate average attendance

How to Run?
python attendance_system.py

 How It Works?
Uses a class AttendanceSystem
Stores student data in a list
Each student has ID, name, course, and attendance

 Learning
Classes & Objects
Lists & Tuples
Loops & Conditions
